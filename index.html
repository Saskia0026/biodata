Selamat datang di web saya!

Halo! Ini adalah website sederhana yang saya buat sebagai bagian dari tugas Praktikum 3 Pemrograman Web. Website ini berisi beberapa halaman seperti profil saya, penjelasan tentang saya, galeri, dan kontak yang bisa dihubungi. Desain ini menggunakan struktur dua kolom, yaitu kolom utama dan kolom samping, dengan kombinasi warna biru pastel, biru muda, dan biru tua.

Tujuan dari website ini adalah untuk melatih kemampuan dasar dalam membuat layout halaman web menggunakan HTML dan CSS, termasuk pembuatan navigasi antar halaman, pengaturan tata letak, dan estetika tampilan web agar terlihat rapi dan menarik.
